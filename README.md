QA Dashboard – Sprint & Release Tracking
🔎 Project Overview:

QA Dashboard is a lightweight web-based reporting tool designed to visualize sprint-level QA metrics such as execution status, defect trends, and progress tracking.

This project was built to strengthen my practical skills in:
Frontend-based reporting dashboards
Data visualization
QA metrics tracking
Real-world sprint reporting concepts

🛠 Tech Stack:
HTML
CSS
JavaScript
Google Sheets Integration (Dynamic Data Source)
GitHub Pages (Deployment)

📌 Key Features:
Dynamic sprint selection
Test case execution tracking (Pass/Fail/Blocked)
Defect summary overview
Date formatting and reporting view
Clean UI with responsive layout
Loader while fetching sprint data
Latest sprint auto-selection logic
